> *"Building the AI-native future: agents that learn, self-improve, and make software teams faster without sacrificing reliability."*

I'm a **Senior AI SDE at Microsoft**, an **AI Agent Developer**, and a **Claude, Cursor, and v0 Ambassador**. I talk about **harness engineering**, AI-assisted coding, and the shift from prompt demos to reliable agent systems.
